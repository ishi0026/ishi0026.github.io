About heading tags

This tag is used for page headings.
You can use the h1 tag for the largest headings.
Others are h2, h3, h4, h5, and h6.
The larger the number, the smaller the words appears.
They help to clarify the cohesiveness of sentences and improve the readability of the page.

About blockquote tags

This tag is used to quote or reprint relatively long passages.
For short sentences, use the q tag for quotations.
Using blockquote, the text will be displayed on a new line and will be prominently displayed.

About strong tags

This tag is an HTML tag that can indicate the importance of text.
The enclosed area will appear in bold and will play in different voices in screen readers;
it is stronger than the em tag; it can be used to indicate the importance of the text; and it can be used to indicate the importance of the text.
It also helps search engines understand the importance of the text.

About br tags

This tag is used to break lines of text.
The "br" stands for Break.
It is used when a line break is necessary due to the content of the text,
not to make space for design purposes.

About hr tags

This tag stands for Horizontal Rule.
It is used for sentence separators and topic conversions.
It is not used as a layout line in design.
